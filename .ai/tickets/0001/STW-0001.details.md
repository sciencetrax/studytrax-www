# STW-0001: Build www.studytrax.com one-shot static site

## Description

One-shot generation of the complete studytrax.com marketing website: 11 static HTML pages, shared CSS/JS, hub SVG asset (or placeholder), `_redirects` file, and README. No build step, no framework. Target audience is Principal Investigators at academic medical centers. Strict content rules apply (no REDCap attacks, no SOC 2 Type II claims, no AI hype, Vanta framed correctly, Tailored Learning Path labeled "in development", Workbench written as mature product, etc.). Full spec lives in `STW-0001.spec.md`.

## Acceptance Criteria

- [ ] All 11 pages exist at the exact paths listed in spec §2
- [ ] Shared `assets/css/site.css` and `assets/js/site.js` exist
- [ ] `_redirects` file exists with required 301 mappings
- [ ] `README.md` exists
- [ ] No forbidden words anywhere in site (unusable, ditch, frustration, overwhelming, unhappy, can't, broken, painful, nightmare, hate)
- [ ] No "SOC 2 Type II" claims; Vanta framed as "continuously monitored through" only
- [ ] No "audited by Vanta" / "Vanta attestation" / "Vanta-certified" / "Vanta penetration test"
- [ ] No present-tense "Studytrax uses AI to…" claims
- [ ] Tailored Learning Path always includes "in development" or "planned capability"
- [ ] Workbench written in mature-product voice (never "planned" / "upcoming" / "beta")
- [ ] Home page opening paragraph verbatim per spec §4
- [ ] Pricing Block 1 copy rendered verbatim per spec §8
- [ ] Compliance/Security canonical Vanta paragraph rendered per spec §9
- [ ] REDCap comparison: validation block (Block 6) ≥ Blocks 2+3+4 combined
- [ ] REDCap comparison CTA preserves load-bearing "IF": "Book a 5-minute fit check — if your REDCap setup is working for what you need, we'll say so."
- [ ] Every page has plain-English first sentence, acronyms spelled out on first use
- [ ] Every image has alt text; skip-to-content link present; semantic HTML5
- [ ] Design tokens defined in `:root` per spec §3
- [ ] All missing items emitted as `<!-- [MISSING: ...] -->` HTML comments + visible `<span class="missing">[detail pending]</span>` placeholders
- [ ] Report delivered: file tree, all [MISSING] markers with page, any §0 rules tempted to break
