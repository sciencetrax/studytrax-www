# STW-0002: Rebuild www.studytrax.com as React web app

## Description

Complete rebuild of the studytrax.com marketing site. Replaces the static HTML site (STW-0001) with a Next.js + React + Tailwind CSS web app. 6 pages (down from 11): Home, Workbench, For Grant Writers, Compare to REDCap, Compliance & Trust, Contact. New content, voice, and structure per the rebuild spec. Full spec in STW-0002.spec.md.

## Acceptance Criteria

- [ ] Next.js + React project scaffolded and builds cleanly
- [ ] All 6 pages exist at correct routes
- [ ] No forbidden words (unusable, ditch, frustration, overwhelming, unhappy, can't, broken, painful, nightmare, hate)
- [ ] No em dashes in copy
- [ ] No nested bullet lists on any page
- [ ] No "SOC 2" claims; Vanta framed correctly
- [ ] AI features: only AI-built forms and AI-assisted data entry in present tense
- [ ] Workbench in mature-product voice
- [ ] No pricing numbers anywhere
- [ ] Testimonials use only provided real names/quotes
- [ ] SEO: title tags, meta descriptions, JSON-LD schemas, OG tags, canonical tags, sitemap.xml, robots.txt
- [ ] Redirects file present
- [ ] Design tokens per spec
- [ ] Mobile-first responsive
- [ ] WCAG 2.2 AA structural compliance
